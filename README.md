https://ereminaolesya.github.io/basefrontend-task5/

## Описание

Реализация  HTML+CSS проекта с плавающей кнопкой, содержащей вложенные кнопки для дополнительных действий.
Задача взята из учебного практического курса.

## Требования к задаче
Позиционирование:
Обязательно использовать комбинацию position: relative и position: absolute для вложенных кнопок
Запрещено использовать display: flex и display: grid
Плавающая кнопка должна быть зафиксирована в углу экрана и не перемещаться в процессе скролла страницы

Интерактивность:
Использовать чекбокс (input[type="checkbox”]) для управления состоянием
Реализация не должна содержать анимаций

Визуальные характеристики:
Не ожидается адаптивность. Тестирование будет происходить на ширине экрана 500px

Функциональность:
Плавающая кнопка должна раскрываться при нажатии на нее

Технические требования:
НЕ использовать display: flex и display: grid
реализовать функциональность при помощи фиксированных, абсолютных и относительных позиционирований
